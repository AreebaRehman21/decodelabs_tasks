# Data Analytics Project 1 - Data Cleaning & Preparation

## Project Overview
This project focuses on cleaning and preparing a raw dataset using Microsoft Excel. The objective is to improve data quality by identifying and handling missing values, removing duplicate records, and correcting inconsistent data formats to ensure the dataset is accurate and ready for analysis.

## Objectives
- Identify and handle missing values
- Remove duplicate records
- Correct inconsistent data formats
- Improve overall data quality
- Prepare the dataset for further analysis

## Tools Used
- Microsoft Excel

## Data Cleaning Steps
- Removed duplicate records
- Checked and handled missing values
- Corrected date, text, and number formats
- Verified data consistency and accuracy

## Skills Demonstrated
- Data Cleaning
- Data Preparation
- Data Validation
- Data Quality Improvement
- Microsoft Excel

## Outcome
The dataset was successfully cleaned and prepared for analysis by ensuring there were no duplicate records, missing values, or formatting inconsistencies.
