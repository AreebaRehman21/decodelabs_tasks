# 📈 Project 2 – Exploratory Data Analysis (EDA)

## 📌 Overview
This project demonstrates Exploratory Data Analysis (EDA) using an E-commerce Sales dataset in Microsoft Excel. The objective was to analyze sales data, calculate descriptive statistics, identify trends, and visualize insights using Pivot Tables and Pivot Charts.

### 🎯 Objectives
- Analyze sales data to discover meaningful insights
- Calculate descriptive statistics
- Identify trends and patterns
- Detect possible outliers
- Present findings using Pivot Tables and Charts

### 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas

### 📂 Dataset Information
- Dataset Type: E-commerce Sales Dataset
- Total Records: 1200 Orders
- Time Period: January 2023 – June 2025

### 📊 Analysis Performed

#### Basic Statistics
- Mean of Total Price: **1053.97**
- Median of Total Price: **823.62**
- Total Orders: **1200**

#### Trend Analysis
- Product-wise Order Analysis using Pivot Table
- Order Status Distribution using Pie Chart
- Monthly Sales Trend using Line Chart

#### Outlier Detection
- Identified unusually high Total Price values
- Maximum Order Value: **3456.40**

### 📌 Key Findings
- Printer was the most ordered product with **181 orders**.
- Cancelled orders had the highest count among all order statuses.
- The average order value was higher than the median, indicating the presence of high-value purchases.
- Monthly sales showed noticeable variations over time.

### 🚀 Skills Demonstrated
- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Pivot Tables
- Pivot Charts
- Data Visualization
- Trend Analysis
